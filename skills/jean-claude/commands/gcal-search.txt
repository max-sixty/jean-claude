Usage: j-claude gcal search [OPTIONS] QUERY

  Search calendar events. Returns JSON with events and optional nextPageToken.

  QUERY: Text to search for in event titles/descriptions

Options:
  --days INTEGER             Days to search (default: 30)
  -n, --max-results INTEGER  Maximum events to return per page
  --page-token TEXT          Token for next page of results
  --help                     Show this message and exit.
