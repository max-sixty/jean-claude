Usage: jean-claude gcal search [OPTIONS] QUERY

  Search calendar events. Returns JSON object: {events: [...], nextPageToken?:
  ...}.

  QUERY: Text to search for in event titles/descriptions

Options:
  --days INTEGER             Days to search (default: 30)
  -n, --max-results INTEGER  Maximum events to return per page
  --page-token TEXT          Token for next page of results
  --calendar TEXT            Calendar ID, email, or name (repeatable; default:
                             primary)
  --help                     Show this message and exit.
