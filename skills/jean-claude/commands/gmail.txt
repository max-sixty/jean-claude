Usage: jean-claude gmail [OPTIONS] COMMAND [ARGS]...

  Gmail CLI - search, draft, and send emails.

Options:
  --help  Show this message and exit.

Commands:
  archive              Archive threads (remove from inbox).
  attachment-download  Download an attachment from a message.
  attachments          List attachments for a message.
  draft                Manage email drafts.
  filter               Manage Gmail filters.
  get                  Get messages by ID, written to files.
  inbox                List threads in inbox.
  labels               List all Gmail labels.
  mark-read            Mark threads as read (all messages in thread).
  mark-unread          Mark threads as unread.
  search               Search Gmail messages.
  star                 Star messages.
  trash                Move threads to trash (all messages in thread).
  unarchive            Move threads back to inbox.
  unstar               Remove star from messages.
