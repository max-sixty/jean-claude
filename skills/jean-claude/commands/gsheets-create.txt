Usage: jean-claude gsheets create [OPTIONS] TITLE

  Create a new spreadsheet.

  TITLE: Title of the new spreadsheet

  Examples:     jean-claude gsheets create "My New Spreadsheet"     jean-
  claude gsheets create "Budget 2025" --sheet "January"

Options:
  --sheet TEXT  Initial sheet name (default: Sheet1)
  --help        Show this message and exit.
