Usage: j-claude gsheets [OPTIONS] COMMAND [ARGS]...

  Google Sheets CLI - read and write spreadsheet data.

Options:
  --help  Show this message and exit.

Commands:
  add-sheet     Add a new sheet to a spreadsheet.
  append        Append rows to a spreadsheet.
  clear         Clear data from a range (keeps formatting).
  create        Create a new spreadsheet.
  delete-sheet  Delete a sheet from a spreadsheet.
  info          Get spreadsheet metadata.
  read          Read data from a spreadsheet.
  sort          Sort data in a range by one or more columns.
  write         Write data to a specific range.
