Usage: j-claude imessage chats [OPTIONS]

  List available iMessage chats.

  Shows chat name (or contact name for 1:1 chats) and chat ID. Use chat ID to
  send to groups.

  Example:     jean-claude imessage chats

Options:
  -n, --max-results INTEGER  Maximum chats to list
  --help                     Show this message and exit.
