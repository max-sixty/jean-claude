Usage: jean-claude imessage [OPTIONS] COMMAND [ARGS]...

  iMessage CLI - send messages and list chats.

  Send via AppleScript (always works). Reading message history requires Full
  Disk Access for the terminal app to query ~/Library/Messages/chat.db.

Options:
  --help  Show this message and exit.

Commands:
  chats         List available iMessage chats.
  mark-read     Mark messages in chats as read by opening them in...
  messages      List messages from local database (requires Full Disk...
  open          Open a chat in Messages.app (marks messages as read).
  participants  List participants of a group chat.
  search        Search message history (requires Full Disk Access).
  send          Send an iMessage to one or more recipients.
  send-file     Send a file attachment via iMessage.
