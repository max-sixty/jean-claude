Usage: jean-claude [OPTIONS] COMMAND [ARGS]...

  jean-claude: Gmail, Calendar, Drive, iMessage, WhatsApp, and Signal
  integration.

Options:
  -v, --verbose    Enable verbose logging to stderr
  --json-log FILE  JSON log file path (default: auto, "-" for stdout, "none"
                   to disable)
  --help           Show this message and exit.

Commands:
  auth         Authenticate with Google APIs.
  completions  Generate shell completion script.
  gcal         Google Calendar CLI - list, create, and search events.
  gdocs        Google Docs CLI - read and write document content.
  gdrive       Google Drive CLI - list, search, and manage files.
  gmail        Gmail CLI - search, draft, and send emails.
  gsheets      Google Sheets CLI - read and write spreadsheet data.
  imessage     iMessage CLI - send messages and list chats.
  reminders    Apple Reminders CLI - create, list, and manage reminders.
  signal       Signal CLI - send messages and list chats.
  status       Show authentication status and API availability.
  whatsapp     WhatsApp CLI - send messages and list chats.
