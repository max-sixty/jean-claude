Usage: j-claude whatsapp send [OPTIONS] [RECIPIENT] [MESSAGE]

  Send a WhatsApp message.

  RECIPIENT: Phone number with country code (e.g., +12025551234) MESSAGE: The
  message text to send

  Use --name to send by contact name instead of phone number. Use --reply-to
  to create a quoted reply to a specific message.

  Examples:     jean-claude whatsapp send "+12025551234" "Hello!"     jean-
  claude whatsapp send --name "John Doe" "Hello!"     jean-claude whatsapp
  send "+12025551234" --reply-to "MSG_ID" "Reply text"

Options:
  --name TEXT      Look up recipient by contact name instead of phone number
  --reply-to TEXT  Message ID to reply to (creates quoted reply)
  --help           Show this message and exit.
